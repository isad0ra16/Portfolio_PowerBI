<h1 data-importer="text" align="left">📊 Power BI</h1>

###

<p data-importer="text" align="left">Este repositório reúne os projetos desenvolvidos durante minha jornada de aprendizado em Power BI, aplicando conceitos de Business Intelligence e Análise de Dados para transformar dados brutos em dashboards interativos e insights estratégicos.<br><br>Ao longo dos projetos, foram utilizadas ferramentas e técnicas como Power Query para extração, limpeza e transformação de dados (ETL), DAX (Data Analysis Expressions) para criação de medidas e indicadores, modelagem de dados com relacionamentos entre tabelas, além da construção de dashboards com foco em usabilidade, storytelling e visualização de dados.<br><br>Cada projeto foi desenvolvido buscando reproduzir cenários reais de negócio, priorizando organização, clareza na comunicação das informações e apoio à tomada de decisão baseada em dados.</p>
